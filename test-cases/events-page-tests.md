# Тест-кейси для сторінки "Події" - GreenCity

## Title 1: Перевірка фільтрації подій за критерієм "Де?" (Локація)
**Preconditions:**
* Користувач знаходиться на сторінці `https://www.greencity.cx.ua/#/greenCity/events`.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Натиснути на випадаючий список фільтра "Де?". | - | З'являється меню з чекбоксами: "Обрати всі", "Онлайн", "Офлайн" та списком доступних міст (наприклад, "Львів"). |
| 2 | Поставити галочку біля чекбоксу з назвою міста. | `Львів` | Фільтр застосовується, сторінка оновлює список знайдених подій. |
| 3 | Перевірити відображені картки подій. | - | На сторінці відображаються лише ті події, які відповідають обраному критерію (відбуваються у місті Львів). |

---

## Title 2: Перевірка переходу на сторінку деталей події
**Preconditions:**
* Користувач знаходиться на сторінці `https://www.greencity.cx.ua/#/greenCity/events`.
* У списку відображається хоча б одна подія.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Натиснути на кнопку "Більше" на картці обраної події. | - | Відбувається перехід на сторінку деталей відповідної події. |
| 2 | Перевірити вміст сторінки деталей. | - | Відображається розгорнута інформація: заголовок, опис, дата/час, кнопки "Зберегти подію" та "Приєднатися до події", а також блок "Коментарі". |

---

## Title 3: Перевірка переходу на форму створення події авторизованим користувачем (Позитивний)
**Preconditions:**
* Користувач **авторизований** у системі (залогінений).
* Користувач знаходиться на сторінці подій.

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Натиснути кнопку "Створити подію" у правому верхньому куті. | - | Відбувається перехід на URL `.../events/create-update-event`. |
| 2 | Перевірити вміст сторінки. | - | Відкривається форма "Створити подію", яка містить поля: "Заголовок" (Введіть назву події*), "Тривалість", "Тематика події" (Економічний, Соціальний, Екологічний) та "Опис подій". |

---

## Title 4: Спроба створення події неавторизованим користувачем (Негативний)
**Preconditions:**
* Користувач знаходиться на сторінці `https://www.greencity.cx.ua/#/greenCity/events`.
* Користувач **не авторизований** у системі (Guest).

| Step | Action | Data | Expected Result |
| :--- | :--- | :--- | :--- |
| 1 | Знайти кнопку "Створити подію" у правій частині сторінки та натиснути на неї. | - | Форма створення події не відкривається. На екрані з'являється модальне вікно для входу / авторизації в системі. |
| 2 | Ввести валідні дані для авторизації та натиснути кнопку входу. | `Email` та `Пароль` | Користувач успішно авторизується, після чого система автоматично перенаправляє його на сторінку "Мій профіль" (замість форми створення події). |
