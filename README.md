# GreenCity Tests

## Короткий опис завдання
Цей репозиторій створено для виконання домашнього завдання з курсів QA. 
Мета роботи — проаналізувати функціональність сторінки подій проєкту GreenCity та застосувати техніки тест-дизайну для створення тестової документації. Репозиторій містить оформлені тест-кейси (позитивні та негативні сценарії), які описують перевірку базового функціоналу сайту.
